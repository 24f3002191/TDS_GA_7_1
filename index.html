Financial Math

**Net Present Value (NPV)**

$$\mathrm{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1+r)^t} - C_0$$

**Compound Annual Growth Rate (CAGR)**

$$\mathrm{CAGR} = \left( \frac{\text{Ending}}{\text{Beginning}} \right)^{\tfrac{1}{n}} - 1$$

Tie back to valuation: use NPV for project appraisal; CAGR for trend communication. Don't over-index on single-period optics.

Code Sample: KPI Snapshot (Python)

from dataclasses import dataclass
from typing import List

@dataclass
class KPI:
    name: str
    value: float
    delta: float  # QoQ change

kpis: List[KPI] = [
    KPI("Revenue (Cr)", 4320, +6.2),
    KPI("Op Margin (%)", 19.4, +0.8),
    KPI("EPS", 18.7, +4.1),
]

for k in kpis:
    arrow = "▲" if k.delta >= 0 else "▼"
    print(f"{k.name:15} {k.value:>8} ({arrow}{abs(k.delta)})")
